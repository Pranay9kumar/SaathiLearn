01 Problem Statement
🎯

Core Problem

Millions of NGO-supported students in India lack structured, trackable, and personalized digital learning — despite having low-cost smartphones.

🧑‍🏫

Mentor Gap

NGO volunteers and mentors have no unified tool to assign content, monitor progress, or give feedback across multiple students.

📊

No Accountability

NGOs can't generate reports on learning outcomes, attendance trends, or skill gaps to prove program impact to donors.

🔋

Engagement Problem

Existing solutions (PDFs, WhatsApp groups) lack motivation loops — students drop off without streaks, rewards, or peer connection.

02 Target Users

Students Primary

  • Age 10–22, Tier 2/3 cities & rural areas
  • Classes 5–12 + vocational learners
  • Low-bandwidth, Android-first users
  • Hindi, Telugu, Marathi + regional languages
  • Limited screen time (1–2 hrs/day)
  • High peer influence — social learning matters

Mentors / Volunteers Secondary

  • NGO field workers & graduate volunteers
  • Part-time commitment (2–4 hrs/week)
  • Non-technical, mobile-first workflow
  • Manage 10–50 students at a time
  • Need quick insight, not complex dashboards
  • Bilingual interface critical

NGO Admins Tertiary

  • Program managers overseeing multiple cohorts
  • Donor reporting — impact metrics, certificates
  • Content curators (upload syllabi, resources)
  • User management — batch enroll, assign mentors

05 User Flow

Student Journey

1

Download & Register

NGO shares a unique invite code. Student signs up with phone number + OTP. Profile created with grade, language, subject preferences.

2

Onboarding Quiz

5-minute baseline assessment to auto-detect current level and generate a personalized learning path.

3

Daily Learning Loop

App shows "Today's Plan" — 2–3 micro-lessons (5 min each). Lesson → mini-quiz → XP earned.

4

Submit Assignments

View mentor-assigned tasks, submit answers (text/photo/voice), receive feedback inline.

5

Ask Doubts

Tap "Ask Doubt" → text or voice message → routed to mentor inbox. Real-time reply notification.

6

Earn & Complete

Complete module → unlock badge → chapter certificate → visible on profile and shareable with family.

Mentor Journey

1

Admin Onboards Mentor

NGO admin creates mentor account, assigns cohort (students group). Mentor gets login link via SMS.

2

View Student Roster

Dashboard shows all students with color-coded activity status: green (active), yellow (2+ days idle), red (at-risk).

3

Assign Content

Select students → pick lesson/quiz from library → assign with due date. Students get push notification.

4

Answer Doubts

Doubt inbox shows unread messages. Reply inline with text, or record a short voice note. Mark as resolved.

5

Track Progress

Weekly auto-generated cohort report. Drill into individual student timelines, quiz scores, and streak data.

6

Export & Report

Generate PDF impact report for NGO admin. Highlights completion rates, top students, avg improvement scores.

08 Gamification Strategy

XP Points System

Complete lesson = +20 XP. Quiz 100% = +50 XP. Daily streak bonus = +10 XP/day. Level up every 500 XP (50 levels total).

Streak Fire

Daily learning streak with fire icons. Breaks after 24 hr gap. Streak shields (earned) can protect 1 miss. Longest streak shown on profile.

Cohort Leaderboard

Weekly XP ranking within class. Top 3 get "Champion" badge. Resets every Monday to keep it competitive for all levels.

Skill Trees

Visual subject trees (Math, Science, English). Unlock next node by completing prior. Visual branching path keeps curiosity alive.

09 UI/UX Design Ideas
🎨

Visual Identity

Deep forest green (#1A6B4A) as primary — trustworthy, growth-oriented. Warm saffron (#F4A024) as energy accent. Illustrated mascot "Saathi" — a friendly owl as companion guide.

📱

Mobile-First Cards

Swipeable lesson cards (Tinder-style swiping for quick daily lessons). Large tap targets (48px+). Bottom navigation bar. Thumb-reachable CTAs.

🌈

Progress Visualization

Circular ring progress per subject. Branching skill tree map with unlocked vs locked nodes. Animated XP bar fill on lesson completion.

🔔

Smart Nudges

Daily reminder at student's preferred time. "Streak at risk!" alert if no lesson by evening. Celebratory confetti animation on badge unlock.

🌐

Accessibility

Full Hindi + regional language UI. Low-data mode (text-only view). High-contrast mode. Voice-over support for visually impaired students.

🎥

Content UX

Auto-play next lesson. Lessons under 7 minutes. Inline quizzes mid-video. Tap-to-pause. Download indicator for offline access. Resume from last position.

👩‍🏫

Mentor UX

One-tap assignment from library. Color-coded student health cards. Quick stats front-and-center (not buried in menus). Voice-reply to doubts.

🏠

Home Screen

Greeting with name + streak. "Today's 3 Tasks" card at top. Recent activity feed. Floating leaderboard widget. One-tap to resume last lesson.

Basic React Native app shell with navigation, theming, and i18n
Student registration + onboarding quiz flow
Deliverable: Working auth + student registration (internal alpha)

Lesson catalog APIs + content upload pipeline (S3 + CloudFront CDN)
Video player with offline download + resume position
Quiz engine — MCQ + short answer + auto-grading
Progress tracking + XP system + streak engine (daily cron job)
Push notifications via Firebase Cloud Messaging
Deliverable: Students can learn + earn XP (beta with 1 NGO)
Month 5–6 Phase 3 — Mentor & Gamification
Mentor web dashboard (Next.js) — student roster, assignment tool, doubt inbox
Real-time doubt messaging (Socket.io) + voice note support
Badges engine — define rules, auto-trigger on milestone events
Leaderboard API + weekly cohort ranking reset
Skill tree visual with lock/unlock states per subject
Deliverable: Full mentor–student interaction loop live

NGO Admin panel — multi-cohort view, bulk enrollment, content management
TimescaleDB analytics pipeline for learning time-series data
PDF impact report generation (donor-facing)

Deliverable: NGO admin can run entire program from one panel
Month 9–12 Phase 5 — Scale & Polish
WhatsApp bot integration for study reminders + quick quiz
AI-based lesson recommendation engine (basic collaborative filtering)
